# Registo de dados do trânsito

[View full project on microbit\.org](https://microbit.org/pt-pt/projects/make-it-code-it/traffic-survey-data-logger)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### O que é isto?

Usa o teu BBC micro:bit para registar dados das coisas que passam à tua frente, como carros, seres vivos ou respostas a perguntas. 

#### Introdução

https://www.youtube.com/watch?v=7Jb-31_72nc

#### Guia do código

https://www.youtube.com/watch?v=2E2i8WjuX-0

Ao carregar num botão, o micro:bit regista o que viste e guarda os dados numa tabela para que os possas analisar mais tarde.

Os dados ficam guardados no teu micro:bit mesmo que desligues as pilhas ou o cabos USB. Podes observá-los mais tarde, quando estiveres no computador.
