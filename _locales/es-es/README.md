# Registrador de datos de tráfico

[Ver proyecto completo en microbit\.org](https://microbit.org/es-es/projects/make-it-code-it/traffic-survey-data-logger)

Para codificar este proyecto tú mismo, haz clic derecho en el fondo del área de trabajo y elige ‘delete all blocks’, luego sigue el video de codificación a continuación. Puedes encontrar los bloques forever y on start en la sección Basic.  
  


### ¿Qué es?

Utiliza tu BBC micro:bit como un registrador de datos para estudiar las cosas que ves, como los diferentes tipos de tráfico, de vida salvaje, o respuestas a preguntas. 

#### Introducción

https://www.youtube.com/watch?v=7Jb-31_72nc

#### Guía de programación

https://www.youtube.com/watch?v=2E2i8WjuX-0

Pulsando un botón, el micro:bit registra lo que has visto y almacena los datos en una tabla para que puedas analizarlos más tarde.

Los datos permanecen en tu micro:bit aunque desconectes las pilas o el cable USB, para que puedas estudiarlos cuando vuelvas a tu ordenador.
